ZADÁNÍ: Landing Page pro mobilní aplikaci ChopChop

Vytvoř responzivní landing page pro mobilní aplikaci ChopChop s mobile-first přístupem.

TECHNICKÉ POŽADAVKY:
- HTML, CSS, vanilla JavaScript (nebo React, pokud je to vhodnější)
- Mobile-first design
- Responzivní pro všechny velikosti obrazovek
- Čistý, moderní kód

BAREVNÁ PALETA:
- Pozadí: #F2F2F2
- Text (černá): #2C2C2C
- Brand color (pro akcenty, podtržení): #A057FF

STRUKTURA PRO MOBILNÍ VERZI:

Header:
- Logo (vlevo)
- Hamburger menu ikona (vpravo)
- Hamburger menu animace: slide-in zprava doleva
- Otevřené menu: fullscreen overlay, obsahuje:
  - Logo (nahoře)
  - Zavírací tlačítko (ikona X)

Hero sekce (vše musí být viditelné bez scrollování):
1. Ikona aplikace (vystředěná)
2. Hlavní headline: "Splitting bills, made simple"
3. Popisný text: "Make every group expense effortless. With ChopChop, you'll always know who owes what—without the hassle, confusion, or endless calculations."
4. Download tlačítka (vystředěná):
   - Obrázek "Download on the App Store"
   - Text "or" mezi tlačítky
   - Obrázek "Get it on Google Play"

ASSETY:
- Všechny obrázky (logo, ikony, mockupy, store tlačítka) jsou dostupné v lokální složce
- Používej relativní cesty k obrázkům

VÝSTUP:
- Vytvoř HTML soubor
- Samostatný CSS soubor pro styly
- JavaScript soubor (pokud potřeba) pro interaktivitu menu
- Struktura složek by měla být čistá a organizovaná

Kód by měl být čitelný, okomentovaný a připravený pro snadné budoucí rozšíření o další sekce.
